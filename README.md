# EP2Xavier

Este repositório foi criado para armazenar o meu segundo e último EP de Estruturas de Dados 2. A ideia principal do EP era implementar um algoritmo que, dado um grafo, determinasse sua "st numeração".

Eu vou ser sincera, eu não faço ideia do que seja essa tal st numeração, e aposto que ninguém além do meu professor sabe, porque o único documento existente na face da Terra sobre essa tal de st numeração é um paper que ele mesmo escreveu. 

Bom, basicamente, nós tivemos que implementar versões diferentes de buscas em profundidade (DFS), então pelo menos esse EP serviu para nós treinarmos um pouco a implementação de algoritmos em grafos.

O trabalho desse professor pode ser encontrado aqui, caso você tenha interesse em entender melhor a st numeração: https://pdfs.semanticscholar.org/37ea/a3291305a49d2153b76aab05ca37d807e334.pdf
